Foor Loops

Ruby For Loop

Basic Ruby for loop:


# Loop over values 0 through 5.

for i in 0..5
prints i
end

//012345

Another way to perform a for loop in Ruby:

(expression).each do |variable[, variable...]| code end

JavaScript For Loops

Example of JavaScript for loop:


for (i = 0; i < cars.length; i++) {
text += cars[i] + "";
}

Operational view of the JavaScript for loop:


for (statement 1; statement 2; statement 3) {
code block to be executed
}
