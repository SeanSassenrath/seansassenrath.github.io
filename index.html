Halle

Halle is an Android & iOS app designed to crowd-source information on parking spaces for drivers in urban areas. The application is built using Ruby on Rails API, JavaScript and jQuery. Mobile integration was done using PhoneGap and jQuery Mobile. Other technologies include Google Maps API, Firebase and Facebook OAuth.

Macy's Guess

Macy's Guess is a game in which a user can try and guess within +/- 20% of an items price. The item's used are all pulled from the ShopStyle api and have been filtered specifically for Macy's merchandise for both men and women. It's built in Foundation 5 for responsive design as well as jQuery for single page app functionality.
